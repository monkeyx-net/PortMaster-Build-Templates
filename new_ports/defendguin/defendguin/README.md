## Controls

| Button | Action |
|--|--|
| Dpad | Move |
| Left Analog | Move|
| Right Analog | Move |
| SELECT | Back |
| START | Start |
| A | Fire |
| B | Smart Bomb |
| X | Boost |
| Y | Reverse |
| L1 | Pause |
